Import the Boiler Room Storefront Theme repo.

After extracting, run:

npm install
npm run audit:theme
npm run audit:auth
npm run audit:finance-export-ui
npm run audit:accounting-storage
npm run audit:accounting-export
npm run audit:billing
npm run audit:customer-ops
npm run audit:technician
npm run audit:service
npm run validate:boundaries
npm run test -- --runInBand
npm run typecheck --workspace @stax/storefront
npm run build --workspace @stax/storefront

Rules:
- Apply the steampunk boiler-room look only to the public ecommerce storefront.
- Do not change command-layer governance.
- Do not add direct Supabase mutation from apps.
- Do not add direct accounting sync.
- Do not add AI diagnosis or autonomous technical recommendations.
- Keep public copy practical and professional.

Return:
- files changed
- command results
- build results
- unresolved blockers
- latest commit hash
