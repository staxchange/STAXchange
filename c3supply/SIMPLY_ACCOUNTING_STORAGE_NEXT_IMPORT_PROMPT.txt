Import the Simply Accounting export storage checkpoint repo.

After extracting, run:

npm install
npm run audit:accounting-storage
npm run audit:accounting-export
npm run audit:billing
npm run audit:customer-ops
npm run audit:technician
npm run audit:service
npm run validate:boundaries
npm run test -- --runInBand
npm run typecheck --workspace @stax/accounting-export-storage
npm run typecheck --workspace @stax/accounting-export
npm run typecheck --workspace @stax/service-billing
npm run typecheck --workspace @stax/commands
npm run typecheck --workspace @stax/admin
npm run build --workspace @stax/admin
npm run build --workspace @stax/storefront

Rules:
- Simply Accounting export storage is finance-only.
- Finance approval is required before signed downloads.
- Signed downloads are for manual accounting review/import only.
- Do not build direct accounting sync.
- Do not auto-post accounting entries.
- Do not auto-approve invoices.
- packages/commands remains the protected mutation gateway.
- Do not commit production secrets.

Return:
- files changed
- command results
- build results
- unresolved blockers
- latest commit hash
