# Steampunk Boiler Room Storefront Theme

The public ecommerce storefront uses a steampunk boiler-room visual language.

## Visual system

- soot / coal background
- brass and copper accents
- riveted cards
- pipe dividers
- pressure-gauge product/category cards
- warm steam-colored typography
- mechanical-room industrial texture

## Customer-facing tone

Use practical, mechanical, treatment-system language:

```txt
boiler-room grade
mechanical room
treatment systems
quote-first
human review
service intake
```

## Boundary

The theme is presentation only.

It does not alter:

- governance
- command-layer mutation routing
- service workflow rules
- finance review rules
- Simply Accounting export prep
- AI/human handoff boundaries
