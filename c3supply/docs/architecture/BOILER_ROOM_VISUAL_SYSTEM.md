# Boiler Room Visual System

This layer adds the product imagery and icon language for the DWG steampunk boiler-room storefront.

## Elements

- pressure gauge icons
- valve icons
- pipe icons
- cartridge icons
- softener vessel icons
- RO membrane icons
- iron/chloramine treatment icons
- service status gauges
- quote workflow mechanism
- mobile boiler nav rivets

## Purpose

The visual system gives the storefront a mechanical, boiler-room feel while keeping product and service language professional.

## Boundary

This is presentation logic only. It does not alter:

- command mutation gateways
- human review workflows
- AI support boundaries
- finance review
- Simply Accounting export prep
